# OneButton Server

  A small backend for [g0v/OneButton][OneButton].

  [OneButton]: https://github.com/g0v/OneButton/

## Run

  1. setup your hub id in the `config.js`

    ```
    module.exports = {
      TYPEFORM: {
        "hub": "all-my-typeforms"
      }
    };
    ```

  2. `npm install`

  3. `npm start`
